# Скрипты интеграции продуктов Dr.Web с сторонними сервисами


## 1. **Интеграция Dr.Web ICAPD c прокси-серверов Squid**  

Данный скрипт проводит интеграцию прокси-сервера Squid с компонентом антивирусной защиты Dr.Web ICAPD в качестве внешнего фильтра.  

Для этого в конфигурационный файл Squid `squid.conf` добавляется блок настроек, перенаправляющий весь трафик на порт компонента Dr.Web ICAPD.   

**Замечания**:  
*Для запуска скрипта необходимо иметь установленный и активированный действующей лицензией Dr.Web Gateway Security Suite.*  

*Интеграция Dr.Web ICAPD и Squid возможна при версии Squid равной или больше 3.0, а также при наличии флага --enable-icap-client при компиляции Squid.*  

*Поддерживаемая версия Python 3.9+*  

*Скрипт работает "из коробки" и не требует установки внешних библиотек*

### Примеры использования

*Команды настройки (`setup`) и удаления (`remove`) требуют прав суперпользователя (`sudo`) для изменения конфигурационных файлов и перезапуска служб.*

#### Показать справку

Чтобы увидеть все доступные команды и параметры, выполните:

```bash
  python3 drweb_squid_integration.py -h
```

#### Настройка интеграции Dr.Web ICAPD c прокси-сервером Squid с настройками по умолчанию.

Команда добавит в конфигурационный файл Squid `squid.conf` блок настроек для работы с Dr.Web ICAPD и настроит параметр ListenAddress компонента Dr.Web ICAPD

```bash
  sudo python3 drweb_squid_integration.py setup
```

#### Удаление интеграции

Команда удалит конфигурационный блок интеграции Squid и ICAPD из файла `squid.conf`.

```bash
  sudo python3 drweb_squid_integration remove
```

### Безопасность

*   Перед каждым изменением конфигурационного файла Squid (squid.conf) создается резервная копия с меткой времени.
*   После внесения изменений и **перед** перезапуском Squid всегда выполняется команда `squid -k parse` для проверки корректности синтаксиса.
*   Команда `remove` удаляет интеграцию ICAPD с прокси-сервером Squid. Флаг `--with-ssl` удалит так же настройки ssl_bump для Squid, если они ранее были добавлены скриптом.
