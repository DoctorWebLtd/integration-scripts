# Утилиты для интеграции продуктов Dr.Web и сторонних сервисов

## Интеграция Dr.Web Gateway Security Suite и Squid

Утилита подключает компонент Dr.Web ICAPD продукта Dr.Web Gateway Security Suite к прокси-серверу Squid в качестве внешнего фильтра.

Для этого в конфигурационный файл Squid добавляются настройки, перенаправляющие весь сетевой трафик на порт компонента Dr.Web ICAPD.

<details>
<summary>Предварительные условия</summary>
  
1. Установленный и активированный действующей лицензией Dr.Web Gateway Security Suite.

2. Squid версии 3.0 или более поздней. Для работы с компонентом Dr.Web ICAPD на этапе компиляции Squid должен быть указан параметр `--enable-icap-client`.

3. Для включения режима `ssl_bump` в Squid на этапе его компиляции должны быть указаны параметры `--enable-ssl-crtd` и `--with-openssl`.

4. Python версии 3.9 или более поздней.

Утилита не требует установки других сторонних зависимостей.
</details>
